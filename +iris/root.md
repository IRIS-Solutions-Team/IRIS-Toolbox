---
title: iris.root
---

{== Current IrisT root folder ==}


## Syntax

    iris.root()
    root = iris.root()


## Output arguments

* `root` [ char | string ] 
> 
> Path to the IrisT root folder.
> 


## Description

The `iris.root()` function is equivalent to the following call to
[`iris.get`]().

```matlab
iris.get('irisRoot')
```
