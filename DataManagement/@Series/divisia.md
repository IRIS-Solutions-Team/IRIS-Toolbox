{== Discrete Divisia index ==}


## Syntax 

    Y = divisia(X,W,RANGE)


## Input arguments 

__`X`__ [ tseries ] 
> 
> Input times series.
> 

__`W`__ [ tseries | numeric ]
> 
> Fixed or time-varying weights on the input
> time series.
> 

__`RANGE`__ [ numeric ]
> 
> Range on which the Divisia index is computed.
> 

## Output arguments 

__`Y`__ [ tseries ]
> 
> Discrete divisia index based on `X` and `W`.
>
