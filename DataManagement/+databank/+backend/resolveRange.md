# `resolveRange`

{== Resolve start and end dates from databank time series ==}


## Syntax 

    [from, to] = resolveRange(inputDb, names, from, to)


## Input arguments 

__`inputDb`__ [ databank ]
> 
> Input databank.
> 

__`names`__ [ string ]
> 
> Variables to be extracted.
> 

__`from`__ [ numeric ]
> 
> The lower range.
> 

__`to`__ [ numeric ]
> 
> The upper range.
> 

## Output arguments 

__`from`__ [ numeric ]
> 
> The lower range.
> 

__`to`__ [ numeric ]
> 
> The upper range.
