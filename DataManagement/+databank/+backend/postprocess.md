# `postprocess`

{== Postprocess a databank ==}


## Syntax 

    db = postprocess(db, names, postprocess)


## Input arguments 

__`db`__ [ databank ]
> 
> Input databank.
> 

__`names`__ [ string ]
> 
> Variables to be extracted
> 

__`postprocess`__ [ string ]
> 
> Type of postprocessing to be done.
> 

## Output arguments 

__`db`__ [ databank ]
> 
> Output databank.
>
