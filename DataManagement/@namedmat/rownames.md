# `rownames`

{== Names of rows in namedmat object ==}


## Syntax 

    rowNames = rownames(X)


## Input arguments 

__`X`__ [ namedmat ]
> 
> A namedmat object (array with named rows and
> columns) returned as output argument from some model functions.
> 


## Output arguments 

__`rowNames`__ [ cellstr ]
> 
> Names of rows in `X`.
>
