# `isempty`

{== True for empty Grouping object ==}


## Syntax

    flag = isempty(g)


## Input arguments

__`g`__ [ Grouping ]
> 
> Grouping object.
> 


## Output arguments

__`flag`__ [ `true` | `false` ]
> 
> True if `g` is a Grouping object with no
> groups.
> 

## Description


## Example

```matlab
g = Grouping();
isempty(g)
ans = 
     1
```
