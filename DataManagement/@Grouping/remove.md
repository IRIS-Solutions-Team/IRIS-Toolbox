# `remove`

{== Remove group from Grouping object ==}


## Syntax

    g = remove(g, groupName1, groupName2, ...)


## Input arguments

* `g` [ Grouping ] - Grouping object.

* `groupName1`, `groupName2`, ... [ char ] - Names of groups that will be
removed.


## Output arguments

* `g` [ Grouping ] - Grouping object.
