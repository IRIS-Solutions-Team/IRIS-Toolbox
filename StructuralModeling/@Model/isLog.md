# `islog`

{== True for empty model object==}


## Syntax 

    Flag = islog(M,Name)


## Input arguments 

`M` [ model ]
> 
> Model object.
> 

`Name` [ char | cellstr ] 
>
> Name or names of model variable(s).
>

## Output arguments 

`Flag` [ `true` | `false` ]
> 
> True for log variables.
>
