# `islog` ^^(Model)^^

{== True for variables declared as log-variables ==}


## Syntax 

    flag = isLog(m, name)


## Input arguments 

__`m`__ [ model ]
> 
> Model object.
> 

__`name`__ [ string ] 
>
> Name or names of model variable(s).
> 

## Output arguments 

`flag` [ `true` | `false` ]
> 
> The output flag is `true` for log variables, `false` otherwise.
>
