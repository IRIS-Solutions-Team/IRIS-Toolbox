# `Window`

{== Clear all charts and settings from the chart ==}

## Syntax

    clear(ch)

## Input arguments

__`ch`__ [ Chartpack ]
>
> Chart object from which all existing charts and settings will
> be cleared.
>

## Description

The `Window` function is useful when you wish to clear the chart window.
